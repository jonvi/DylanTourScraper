
# The Bob Dylan Tour Scraper
### Description
The purpose of the program is to scrape the tour date, location, venue and ticket link.
Then format the information to a subreddit sidebar.

### How to Run the Scraper
Simply execute bobrunner.py with Python3 and [BeautifulSoup4](https://www.crummy.com/software/BeautifulSoup/bs4/doc/) installed.
